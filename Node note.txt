The top most dir in file system in root dir and with '/'. It is the full path to reach the file which is also called absolute path. It is the every step necessary of filesystem to reach the file. For understanding remember absolute path start with / or \ depending upon the platform. 

It is the path from the current working directory(CWD). It is faster than absolute path since the path has less steps to follow. It is more easier and clean to use relative path, consider it more than absolute path.
